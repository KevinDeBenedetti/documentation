## Frameworks

<CardGrid>
    <Card title="Vue" icon="seti:vue">
        Vue.js est un framework JavaScript progressif pour la construction d’interfaces utilisateurs. Il est simple à utiliser, flexible et se distingue par sa capacité à être intégré progressivement dans des projets existants.
        <LinkCard title="Voir plus" href="/documentation/vue/" />
    </Card>
    <Card title="Nuxt" icon="node">
        Nuxt.js est un framework basé sur Vue.js, orienté vers le développement d’applications universelles et optimisées pour le SEO. Il simplifie la gestion du rendu côté serveur (SSR) et offre une excellente structure pour le développement moderne.
        <LinkCard title="Voir plus" href="/documentation/nuxt/" />
    </Card>
    <Card title="Symfony" icon="seti:php">
        Symfony est un framework PHP populaire et robuste, idéal pour développer des applications web évolutives. Il met l’accent sur la réutilisation de composants et suit les meilleures pratiques de l’industrie.
        <LinkCard title="Voir plus" href="/documentation/symfony/" />
    </Card>
    <Card title="Fast API" icon="seti:python">
        FastAPI est un framework Python conçu pour créer des APIs rapides et performantes avec une validation automatique des données et un support natif pour les types de données.
        <LinkCard title="Voir plus" href="/documentation/fast-api/" />
    </Card>
    <Card title="Django" icon="seti:python">
        Django est un framework Python de haut niveau, conçu pour le développement rapide d’applications web. Il fournit une interface d’administration intégrée, un ORM puissant et une structure modulaire pour des projets robustes.
        <LinkCard title="Voir plus" href="/documentation/django/" />
    </Card>
</CardGrid>

## CI / CD

<CardGrid>
    <Card title="GitHub" icon="seti:github">
        GitHub est une plateforme de développement collaborative, utilisée pour l’hébergement et la gestion du code source avec Git. Elle permet l’intégration continue (CI) et le déploiement continu (CD) pour une automatisation fluide des projets.
        <LinkCard title="Voir plus" href="/documentation/github/" />
    </Card>
    <Card title="Docker" icon="seti:docker">
        Docker est une technologie de conteneurisation qui permet aux développeurs de packager leurs applications avec toutes leurs dépendances, garantissant une exécution cohérente dans différents environnements.
        <LinkCard title="Voir plus" href="/documentation/docker/" />
    </Card>
    <Card title="Hébergement" icon="seti:db">
        L’hébergement regroupe les solutions permettant de rendre des applications accessibles en ligne, allant des serveurs partagés aux solutions cloud modernes, en passant par des configurations dédiées.
        <LinkCard title="Voir plus" href="/documentation/hosting/" />
    </Card>
</CardGrid>

## Systémes d'exploitation

<CardGrid>
    <Card title="MacOS" icon="apple">
        MacOS est le système d’exploitation développé par Apple, connu pour sa stabilité et son environnement propice au développement, notamment grâce à son noyau Unix.
        <LinkCard title="Voir plus" href="/documentation/macos/" />
    </Card>
    <Card title="Debian" icon="linux">
        Debian est une distribution Linux stable et polyvalente, reconnue pour sa sécurité et sa flexibilité, souvent utilisée pour des serveurs et des environnements de développement.
        <LinkCard title="Voir plus" href="/documentation/debian/" />
    </Card>
    <Card title="Ubuntu" icon="linux">
        Ubuntu est une distribution Linux dérivée de Debian, connue pour sa simplicité d’utilisation et son large soutien communautaire, idéale pour les développeurs et les utilisateurs de bureaux.
        <LinkCard title="Voir plus" href="/documentation/ubuntu/" />
    </Card>
