Nuxt.js est un framework Vue.

## Installer Nuxt

Lancer l'installation du projet.
```shell frame="none"
npx nuxi@latest init <nom-du-projet>
```
Naviguer dans le répertoire.
```shell frame="none"
cd <nom-du-projet>
```
Installer les dépendances, package.json.
```shell frame="none"
npm install
```
Lancer le projet.
```shell frame="none"
npm run dev
```
