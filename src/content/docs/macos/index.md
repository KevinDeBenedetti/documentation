:::tip[Le saviez-vous ?]
[MacOS](https://fr.wikipedia.org/wiki/MacOS) est un système d’exploitation partiellement propriétaire développé et commercialisé par Apple depuis 1998, dont les versions les plus récentes sont macOS Sonoma, macOS Ventura lancée le 24 octobre 2022 et macOS Monterey lancée le 25 octobre 2021 pour le grand public.
:::
