## Configurer son projet

<Steps>
1. ### Installer `python-dotenv`

    ```sh
    pip install python-dotenv
    ```

2. ### Configuration

    Modifier le fichier `settings.py`.

    ```diff lang="python"
    // settings.py

    +import os
    + from dotenv import load_dotenv

    +load_dotenv()

    -SECRET_KEY = 'secret-key'
    +SECRET_KEY = os.environ['SECRET_KEY']
    ```

3. ### .env

    Ajout d'un fichier `.env` à la racine du projet, au même niveau que `manage.py`. 

    ```env
    SECRET_KEY=<secrete-key-of-the-project>
    DEBUG=True
    ```
</Steps>
