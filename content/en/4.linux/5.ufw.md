## Install & enable

```bash
sudo systemctl start fail2ban
```
Start the Fail2Ban service.

```bash
sudo ufw enable
```
Enable the UFW firewall.

```bash
sudo apt install ufw
```
Install UFW.

```bash
sudo ufw status
```
Check UFW status.

## Basic commands

```bash title="Check status"
sudo systemctl status fail2ban
```

```bash title="Restart configuration"
sudo systemctl restart fail2ban
```

## Configure the firewall

```bash
sudo ufw default deny incoming
```
Default policy: deny incoming connections.

```bash
sudo ufw default allow outgoing
```
Default policy: allow outgoing connections.

```bash
sudo ufw allow ssh
```
Allow SSH connections (port 22 by default).

```bash
sudo ufw allow http
```
Allow HTTP connections (port 80).

```bash
sudo ufw allow https
```
Allow HTTPS connections (port 443).
