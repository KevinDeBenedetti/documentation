## Domains

::steps{level="3"}

### Modify DNS Records

Modify the `A` record, add the VPS IP address (`IPv4` and `IPv6`).

Modify the `data` field: VPS IP address.

### Add the Domain Name

Add a domain in the Secondary DNS section.

Follow the instructions to add the domain to a secondary DNS.

::

::note
This information is provided by the domain provider and/or hosting provider.
::

### Subdomains

Add a subdomain in the DNS section.

::steps{level="3"}

### Add a CNAME Record

`Name` corresponds to the subdomain, and `Data` corresponds to the domain name.

::

::tip
[Godaddy Documentation](https://www.godaddy.com/help/add-a-cname-record-19236)
::

## Dokploy

Dokploy is an open-source continuous deployment tool, written in Python, that automates application deployment on remote servers via SSH. It is designed to be simple to use and configure, while offering great flexibility to adapt to different environments and deployment needs.

### Telegram Notifications

To configure Telegram notifications, add the following information to the `dokploy.yml` file:

1. Name: enter a name for the bot.

2. Bot Token: enter the bot token. Available on the `@BotFather` Telegram app.

3. Chat ID: enter the chat ID.
   To retrieve the channel ID, go to `https://api.telegram.org/bot<BOT_TOKEN>/getUpdates`

::note

- [GitHub](https://gist.github.com/nafiesl/4ad622f344cd1dc3bb1ecbe468ff9f8a)
