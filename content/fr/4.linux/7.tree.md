## Installer `tree`

| Système / OS                                    | Installation / disponibilité                                                                |
| ----------------------------------------------- | ------------------------------------------------------------------------------------------- |
| **macOS**                                       | via Homebrew : `brew install tree`                                                          |
| **Linux / Unix** (Debian, Ubuntu, Fedora, etc.) | via le gestionnaire de paquets : ex. `sudo apt-get install tree` ou `sudo yum install tree` |
| **Windows** (invite de commandes / cmd.exe)     | la commande `tree` est présente nativement dans l’invite de commandes                       |
| **Autres Unix / BSD**                           | souvent disponible via le gestionnaire de paquets ou en compilant l’outil                   |

## Commande `tree`

La commande `tree` permet d’afficher en console l’arborescence d’un répertoire de façon lisible (sous-dossiers, fichiers, profondeur, etc.).
Elle est disponible sur plusieurs systèmes d’exploitation :

### Exemple d’utilisation

```sh
# arborescence simple (profondeur 3 max)
tree -L 3

# incluant fichiers cachés, en excluant patterns (ex.: venv, .git)
tree -L 3 -a -I 'venv|.git|__pycache__'

# avec tailles lisibles et dates
tree -L 3 -a -h -D -I 'venv|.git|__pycache__'

# uniquement les dossiers (profondeur 2)
tree -d -L 2

# afficher les chemins complets
tree -f -L 3

# rediriger la sortie vers un fichier
tree -L 4 -a -I 'venv|.git|__pycache__' > tree.txt
```
